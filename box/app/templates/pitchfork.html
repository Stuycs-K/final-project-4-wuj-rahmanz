Pitchfork



How does pitchfork work?

Pitchfork takes the nth index in each payload.

Examples:
  • Request 1: Username = payload1[0] Password = payload2[0]
  • Request 2: Username = payload1[1] Password = payload2[1]
  • Request 3: Username = payload1[2] Password = payload2[2]

How many requests does it make?

Pitchfork will make as many requests as the length of the shortest payload.

What is pitchfork useful for?

Very useful when an attack needs 2 different but related parameters. For example, the username and password in the examples above.
