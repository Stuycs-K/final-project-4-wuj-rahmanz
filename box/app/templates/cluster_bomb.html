Cluster Bomb



How does cluster bomb work?

Cluster bomb is similar to pitchfork except it runs all possible combinations of the payloads inputted.

Examples:
  • Request 1: Username = payload1[0] Password = payload2[0]
  • Request 2: Username = payload1[0] Password = payload2[1]
  • Request 3: Username = payload1[0] Password = payload2[2]

How many requests does it make?

Clusterbomb will make payload1.length * payload2.length requests.

What is cluster bomb useful for?

Cluster bomb is the brute force attempt of pitchfork. If the inputs are completely unrelated, you use cluster bomb and pray something works out.
