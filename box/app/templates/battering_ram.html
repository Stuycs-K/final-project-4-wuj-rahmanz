Battering Ram

How does battering ram work?

Battering ram takes a word from payload, inputs them in all parameters, and continues until there is a result.

How many requests does it make?

Battering ram will make payload.length * parameters requests.

What is battering ram useful for?

You'll use battering ram rarely, but it's useful for when you have to input the same thing in multiple parameters.
